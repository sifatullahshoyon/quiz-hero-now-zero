What is the difference between Local Storage, Secession Storage & Cookies?

localStorage is a browser's memory, where small data can be stored, when a user or customer can bookmark some products of that website without logging in to a website, then all the information of that product is actually stored in local storage, because The user has not given any of his information to the website, so none of the customer's or user's information can be kept in the database and as a result, it cannot be understood about that customer, so all his information is saved in the memory of the browser called local storage, so that the When the customer comes to the website again through the same browser of the device, he can see all the information again, all the information that he has bookmarked or edited in the previous time. If something is saved in local storage, it remains in that browser until the user deletes all that information.
